# 🌍 AI Language Translator

An AI-powered Language Translation Web Application built using Flask, HTML, CSS, and JavaScript as part of the CodeAlpha Artificial Intelligence Internship.

---

## 🚀 Features

- Translate text between multiple languages
- Auto language detection
- Modern responsive UI
- Copy translated text
- Text-to-speech support
- Translation history
- Loading animation
- Mobile-friendly design

---

## 🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- deep-translator
